# plsm-GD

project-level session memory for Google Docs

---

**friction**: after every browser restart, manually reopen every file editors that are tied to a single project.  

**goal**: restore working context instantly by defining a project only once and everything relevant reopened in one action.  

**core idea**: group docs into project-level sets; opening one auto-restores the working set into a single Chrome tab group.

**install**: Chrome extension (link).

---

**features**:
- adding the current Google Doc/Sheet/Slide to a named group
- adding selected files from Drive or Docs Home to a group
- adding entire Drive folders as dynamic groups
- updating existing groups or creating a new group
